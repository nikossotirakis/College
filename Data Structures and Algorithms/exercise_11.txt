Η παράλληλη δυαδική αναζήτηση έχει καλύτερη απόδοση σε σύγκριση με την παραδοσιακή δυαδική αναζήτηση, 
γιατί χρησιμοποιεί πολλαπλούς πυρήνες για την παράλληλη αναζήτηση του πίνακα. Αυτό την καθιστά μια εξαιρετικά 
αποτελεσματική μέθοδο για την αναζήτηση δεδομένων σε μεγάλες βάσεις.

-- STEPS --

    1. Ταξινόμηση του πίνακα

    2. Διαίρεση του πίνακα στα δύο, ξανά και ξανά, μέχρι να χρησιμοποιηθούν όλοι οι πυρήνες της CPU

    3. Εκτέλεση δυαδικής αναζήτησης σε κάθε υποτμήμα του πίνακα

    4. Μόλις ολοκληρωθεί η αναζήτηση σε όλα τα τμήματα, τα αποτελέσματα συνδυάζονται για να βρεθεί η τελική θέση του στοιχείου (End to Start approach)
